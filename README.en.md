# 基于SpringBoot+Vue的健身管理系统

#### Description
该系统用于帮助用户记录和管理他们的健身活动，包括运动计划、卡路里消耗、锻炼时间、体重管理等。系统通过友好的用户界面和实时数据交互，展示用户的健身进度，支持用户根据个人目标制定健身计划。
技术栈：SpringBoot+Mybatis-Plus+MySQL+Vue3+Vuetify+Echarts+SSE。
